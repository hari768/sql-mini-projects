# Amazon E-Commerce Analytics

## Project Overview

This project simulates a real-world Amazon E-Commerce database and demonstrates how SQL JOIN operations are used to analyze customer purchases, products, categories, and orders.

---

## Database Name

amazon_ecommerce_analytics

---

## Tables Used

### 1. Customers
Stores customer information.

### 2. Categories
Stores product categories.

### 3. Products
Stores product details and category relationships.

### 4. Orders
Stores customer purchase transactions.

---

## Concepts Covered

- Primary Key
- Foreign Key
- INNER JOIN
- LEFT JOIN
- RIGHT JOIN
- FULL OUTER JOIN
- Table Aliases
- Multi-Table Joins
- Join with Filters
- Real-World Database Design

---

## Business Questions Solved

### Task 1
Show customer order details.

### Task 2
Show products with their categories.

### Task 3
Show customer purchase analysis.

### Task 4
Generate a complete business report using 4 tables.

### Task 5
Show all customers even if they have no orders.

### Task 6
Show all orders even if customer data is missing.

### Task 7
Compare complete customer and order records.

### Task 8
Analyze Gold Membership customers.

### Task 9
Identify products above ₹10,000.

### Task 10
Show Electronics category products.
